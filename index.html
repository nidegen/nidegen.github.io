My First Usage of Github

I got introduced to GitHub during my master thesis at the Autonomous System Lab where I worked under Simon Lynen (@slynen) and Titus Cieslewski (@tcies) for a multiagent mapping framework. We thereby also used Google's Project Tango devices for data acquisition. I enjoy every aspect of GitHub and predict a big future:). Please contribute to my site, see this section.

Contribute

If you want to contribute please open a branch named contribution/< github username> and a PR named after what it does.

$ cd your_repo_root/repo_name
$ git fetch origin
$ git checkout contribution/< github username>
